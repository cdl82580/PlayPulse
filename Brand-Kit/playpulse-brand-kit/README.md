# PlayPulse — Brand Asset Kit

Modern, vector-first assets generated from the PlayPulse brand guide.
Every logo is built as a true vector (the wordmark is outlined, so it renders
identically with or without fonts installed) plus high-res PNGs for convenience.

Open **playpulse-brand-sheet.html** in any browser for the full living style sheet
(logos, app icons, color, type) — it's fully self-contained, no internet needed.

## Files

### /logos
- `playpulse-logo-horizontal-{dark,light,trans}.svg` — mark + wordmark, side by side
- `playpulse-logo-stacked-{dark,light,trans}.svg` — primary lockup, mark over wordmark
- `playpulse-wordmark-{light,dark}.svg` — "PlayPulse" type only
- `playpulse-logomark.svg` / `.png` — the play-into-pulse mark (full color)
- `playpulse-logomark-white.svg` / `.png` — reversed, for busy/dark grounds
- `playpulse-logomark-mono.svg` — single-color ink version
- `*-trans` = transparent background · `*.png` = 1024–1600px raster

### /app-icons
- `playpulse-app-icon.svg` + PNGs at 1024 / 512 / 256 / 180 / 128 / 64 px
- `playpulse-favicon.svg` + `playpulse-favicon-512.png` (circular)

### /social
- `playpulse-social-banner.svg` / `.png` — 1500×500 cover (LinkedIn/X/profile)

### /fonts
- Sora (Regular, Medium, SemiBold, Bold, ExtraBold) + `OFL.txt` license

## Color

Primary gradient: #7B2DFF → #FF2D72 → #FF8A00 → #FFC700
Secondary: #2A1B4D · #2563EB · #00CB96 · #FFB703 · #F1F5F9
Neutrals: #0B0D17 · #131722 · #1E2433 · #4B5563 · #9CA3AF · #E5E7EB · #FFFFFF

Reserve the gradient for the mark, primary actions, and accents. Let neutrals
carry layout and body text.

## Type

Sora — geometric grotesque. Headlines lean oblique/italic; data and captions
stay upright. Weights in use: 400 / 500 / 600 / 800.

## Notes
- SVGs are editable in Figma, Illustrator, Inkscape, or any code editor.
- The mark and wordmark share one userSpace gradient so the color flows
  continuously across the whole lockup.
- Sora is licensed under the SIL Open Font License (see fonts/OFL.txt).
